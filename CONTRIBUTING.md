# Contributing

Thanks for your interest in improving TLDV Video Downloader — contributions are welcome.

This is a free, MIT-licensed utility. Issues, feature requests, and pull requests are all appreciated.

## How to contribute

1. **Open an issue** first for anything non-trivial, so we can discuss the approach.
2. **Fork** the repo and create a branch off `main`.
3. Keep changes focused — one logical change per PR.
4. Make sure the script still runs (`python tldv_downloader.py`) and CI is green.
5. **Open a pull request** describing what changed and why.

## Ground rules

- Be respectful in issues and reviews.
- Use this tool responsibly and in accordance with tldv.io's terms of service.
